# Archived Legacy Assets

This folder contains the previous AWS/Jenkins-era implementation that has been preserved for reference.

- `terraform/` - original EC2 and EKS Terraform configuration
- `pipeline_script/` - original Jenkins pipelines for build, deploy, and cleanup

These files are kept as archive material only. The active delivery path now uses GitHub Actions, Docker Hub, k3s, ArgoCD, SonarCloud, Trivy, Prometheus, and Grafana.
